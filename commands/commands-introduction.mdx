LondonTransit offers a wide range of commands to help you navigate London's public transport system. This guide provides an overview of the available commands and how to use them effectively.

## Command Syntax

Commands in LondonTransit are slash commands, making them easy to view and use either by typing `/` in the chat or by clicking the commands / apps button in the chat bar.
